# **App Name**: PillPal Reminder

## Core Features:

- Medication Schedule: Create and manage medication schedules with specific times and dosages.
- Customizable Alarms: Set alarms for each medication, which trigger a reminder notification.
- Text-to-Speech Alarm: Convert medication details (name, dosage) into speech for audio reminders.
- Multilingual Support: Offer the app in multiple languages, adapting text and voice prompts.
- Snooze Function: Implement a snooze option for alarms to delay reminders.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a sense of calm and reliability.
- Background color: Very light lavender (#F5F5FF) for a clean and gentle interface.
- Accent color: Muted blue (#778899) to provide subtle highlights and indicate interactive elements without causing distraction.
- Headline font: 'PT Sans' for clear and friendly headlines.
- Body font: 'Inter', for readable and neutral body text.
- Use clear, minimalist icons to represent medication types and reminder settings.
- Subtle transitions and animations to provide visual feedback and a smooth user experience.
